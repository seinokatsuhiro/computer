iWork
==================================================================


2013.11.2 に Keynote をインストールしようとしたが、
新 OS の Mavericks (10.9) 発売以降、Mountain Lion (10.8) 向けの
Keynote を App Store からダウンロードできないらしい。
具体的には、App Store で Keynote をダウンロードすると、
Mavericks のダウンロードが始まってしまう。
しかたなく、ダウンロード後に Mavericks のインストーラを削除して、
ひとつ前の Keynote を入れることに。

Pages、Numbers、Keynote を含む iWork 09 のインストール DVD を
4 年前に購入していたので、これを古い Macbook Pro 経由での
DVD 共有でインストールする。
念のため、`cp -pr` でインストール DVD から、
以下の内容のファイルをコピーしておく。

    -rw-r--r--@ 1 ...      0 12 21  2008 Install iWork '09
    drwxr-xr-x@ 3 ...    102 12 21  2008 Packages/
    drwxr-xr-x  3 ...    102 12 21  2008 Read Before You Install iWork.app/
    -rw-r--r--@ 1 ...  64630 11 22  2008 iWorkBackground.tiff

このインストーラ `Install iWork '09` を起動して、
インストール後、App Store で [アップデート][iWork-9.3] し、
Keynote 5.3 になった。

Pages と Numbers は、あまり、使わない。
ただ、関係データのエディタと計算機があれば、使いたい。



[iWork-9.3]: http://support.apple.com/kb/HT5534?viewlocale=ja_JP
